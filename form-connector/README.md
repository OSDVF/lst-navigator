# form-connector

To install dependencies:

```bash
bun install
```

To build:

```bash
bun run build
```

To publish to Google Apps Script project:

```bash
bun push
```

## Setup
These [Script Properties](https://stackoverflow.com/a/72008088) must be set
```
apiKey
projectId
appId
```
